# Python Website Title Scraper

A simple command-line tool that extracts the title of a website using Python.

This project demonstrates basic web scraping, HTTP requests, and HTML parsing using Python.

## Features

- Fetches the HTML content of a website
- Extracts and prints the page title
- Handles invalid URLs and connection errors
- Uses requests and BeautifulSoup for web scraping

## Technologies

- Python 3
- requests
- BeautifulSoup (bs4)
